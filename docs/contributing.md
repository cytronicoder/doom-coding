# Contributing

Contributions are welcome! If you have found ways to improve this workflow, please share them. We are specifically looking for:

- Better mobile terminal configurations
- Safer defaults for remote access
- Smoother preview workflows for web apps
- Reliable "always-on" host hardware/software setups
- Hardening steps that reduce risk without adding significant friction

## How to Contribute

1. Open an issue for bugs or suggestions
2. Submit a PR for direct improvements to the documentation
